GIT_BASH

Commandes et Procédures

note : la console GIT_BASH s'opère comme une console LINUX

COMMANDES

2 types de commandes :

  • les commandes standard et les commandes GIT (pour toutes actions dans de synchronisation dans GIT)
  • les commandes GIT sont toujours précédées de 'git'.
COMMANDE STANDARD
utiliser la touche 'inser' coller (ctrl + v)
pwd Connaitre le répertoire actuel
cd nomDeRépertoire / nomDeSousRépertoire allez au sous répertoire
cd .. remonter d'un répertoire
ls liste des fichiers du répertoire
mkdir nomNouveauRépertoire créer un répertoire
touch nomFichier.txt créer un nouveau fichier nomFichier.txt
rm nomFichier.txt supprimer le fichier nomFichier.txt
brackets nomFichier.txt ouvrir le fichier nomFichier.txt avec Brackets
